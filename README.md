# Image tagging

## To install

* Install node.js and npm
* Run ``npm install``

## To run

* Edit main.js and change images_dir to point to a directory containing images.
* Run ``npm start``

## Key bindings

| Action         | Key   |
| -------------- | ----- |
| Tag as "good"  | ``1`` |
| Tag as "bad"   | ``2`` |
| Next image     | ``l`` |
| Previous image | ``h`` |
| Save           | ``s`` |
